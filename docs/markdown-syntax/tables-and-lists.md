# Tables

Tables get the responsive CSS class automatically, so wide tables scroll horizontally on small screens instead of breaking the layout.

---

## Simple Table

| Method      | Return | Description                  |
|-------------|--------|------------------------------|
| `exists`    | bool   | Check if the data is cached  |
| `set`       | bool   | Save data to cache           |
| `get`       | mixed  | Get data by key              |
| `remember`  | mixed  | Get or store data            |
| `delete`    | bool   | Delete data by key           |

## Alignment

| Left aligned | Centered | Right aligned |
|:-------------|:--------:|--------------:|
| apple        |   red    |          1.20 |
| banana       |  yellow  |          0.80 |
| blueberry    |   blue   |          3.50 |
