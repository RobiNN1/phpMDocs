# Text Formatting

Basic text formatting including accented characters like ľ, š, č, ť, ž, ý, á, í, é that must work in headings, descriptions and search results without any problem or limitation.

---

## Emphasis

Text can be **bold**, *italic*, ***bold and italic*** or ~~strikethrough~~.
Inline `code` is wrapped in backticks.
